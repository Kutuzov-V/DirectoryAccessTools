Программа для сбора информации о списках доступа предназначена для упрощения процессов мониторинга доступа к  корпоративным информационным ресурсам. Для выполнения своих функций она использует данные полученные с LDAP сервера (Active directory) и файловой системы Windows. Доступ к LDAP каталогу необходим для отображения в отчёте ФИО пользователя, за которым закреплена учётная запись, фигурирующая в списках доступа к каталогу (ACL - списках).

!!!ВАЖНО
- Для формирования матриц доступа (или списков доступа в зависимости от требуемой формы отчёта) необходимо, чтобы уровень прав пользователя, запускающего данную программу позволял читать ACL к указанному каталогу.

- Для получения ФИО пользователя учётной записи требуется, используемая учётная запись (вводится отдельно) имела право на чтение LDAP-каталогов
